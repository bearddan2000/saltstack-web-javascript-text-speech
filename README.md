# saltstack-web-javascript-text-speech

## Description
A small project to demonstrate a browser
feature that turns text to speach. Salt
was used to build the apache server.

## Tech stack
- salt
- jquary 3.5

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
